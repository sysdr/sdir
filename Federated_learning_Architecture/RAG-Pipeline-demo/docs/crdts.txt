Conflict-free Replicated Data Types are data structures that can be updated independently on multiple replicas and merged without coordination or conflict. The mathematical property that enables this is convergence: regardless of the order in which updates are applied, all replicas reach the same state once they exchange information. G-Counters are the simplest CRDT: each node maintains its own counter; the merged value is the maximum per-node. PN-Counters extend this with separate increment and decrement vectors. LWW-Element-Sets use timestamps to resolve add and remove conflicts by keeping the most recent operation. Riak's eventual consistency model was built around CRDTs. Redis includes several CRDT-based data types for active-active geo-replication.
